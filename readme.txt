This repository contains a HTML5 game which use Phaser as frame work and PlayFab as backend platform.
This game based on farm management simulation, its main modules are as follows:
	• Basic operations: 
		○ Sow
		○ Fertilize
		○ Harvest
		○ Eradicate
	• Economic system:
		○ Purchase seeds and fertilizers
		○ Sell agricultural products
		○ Get random items by unlocking treasure chests.
	• Reward system:
		○ new user package
		○ daily login reward
